# Sudoku

A number placement puzzle built with Next.js, React, and Tailwind CSS.

![Sudoku](public/favicon.svg)

## Features

- **4 difficulty levels** — Easy, Medium, Hard, Expert
- **Notes mode** — pencil in candidate numbers per cell
- **Hints** — reveal the correct digit for the selected cell
- **Auto-solve** — fill the remaining board instantly
- **Undo** — step back through your move history
- **Mistake counter** — tracks incorrect placements
- **Stopwatch** — times each puzzle
- **Conflict highlighting** — red cells show row/column/box violations
- **Keyboard support** — navigate and play without a mouse

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `←↑→↓` | Navigate cells |
| `1–9` | Place number (press again to remove) |
| `Backspace` / `Delete` / `0` | Erase cell |
| `N` | Toggle notes mode |
| `Ctrl+Z` | Undo last move |

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- [Next.js 16](https://nextjs.org/) — App Router, static export
- [React 19](https://react.dev/)
- [Tailwind CSS v4](https://tailwindcss.com/)
- TypeScript

## Algorithm

Puzzles are generated by:
1. Filling a blank board with a valid solution via backtracking + random shuffle
2. Removing cells one-by-one while verifying the puzzle retains a **unique solution**

Difficulty is controlled by the number of clues left:

| Difficulty | Clues |
|------------|-------|
| Easy | 40 |
| Medium | 32 |
| Hard | 26 |
| Expert | 22 |
